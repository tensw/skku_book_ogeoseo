## 1. 관리자 기능 개요

### 1.1 관리자 권한

관리자는 다음 기능에 접근할 수 있습니다:

| 기능 | 설명 |
|------|------|
| **홈페이지 관리** | 배너 수정, 이달의 책 관리 링크 |
| **프로그램 관리** | 이달의 책, 주차별 도서 배정 |
| **독토 관리** | 독토 생성/수정/삭제 |
| **공지사항 관리** | 공지 생성/수정/삭제 |
| **서평 관리** | 서평 검토/삭제 |
| **고전 100선 관리** | 도서 목록 관리 |

### 1.2 관리자 메뉴 접근

관리자로 로그인하면:
- 홈페이지에 **관리자 전용 버튼**이 표시됩니다.
- 각 카드에 **수정/삭제** 아이콘이 나타납니다.
- 하단에 **관리 페이지 바로가기** 버튼이 표시됩니다.

### 1.3 데이터 연동

v2.0부터 **SharedDataContext**를 통해 데이터가 실시간 연동됩니다:
- 관리자가 수정한 내용이 모든 페이지에 즉시 반영됩니다.
- 홈페이지, 독모, 독토, Talk 페이지가 동기화됩니다.
- 사용자의 신청 현황이 내 서재에 자동 반영됩니다.

## 2. 관리자 로그인

### 2.1 로그인 방법

1. 오거서 메인 페이지에 접속합니다.
2. 우측 상단의 프로필 아이콘을 클릭합니다.
3. **관리자 로그인** 옵션을 선택합니다.
4. 관리자 계정으로 로그인합니다.

### 2.2 관리자 계정

- 관리자 계정은 시스템 관리자에게 문의하여 발급받습니다.
- 보안을 위해 비밀번호를 정기적으로 변경해주세요.

### 2.3 관리자 모드 확인

로그인 후 다음을 확인하세요:
- 홈페이지에 관리자 버튼 (톱니바퀴 아이콘) 표시
- 카드에 수정/삭제 아이콘 표시
- 하단 우측에 "관리" 플로팅 버튼 표시

## 3. 홈페이지 관리

### 3.1 배너 수정

관리자는 홈페이지 상단 배너를 수정할 수 있습니다.

#### 수정 방법
1. 홈페이지에서 배너 영역의 **수정 아이콘**을 클릭합니다.
2. 배너 수정 모달에서 다음을 수정합니다:
   - **이미지 URL**: 배경 이미지 주소
   - **제목**: 메인 슬로건
   - **부제목**: 서브 메시지
3. "저장" 버튼을 클릭합니다.

#### 권장 이미지 크기
- 데스크톱: 1200 x 600px
- 모바일: 800 x 400px
- 형식: JPG, PNG, WebP

### 3.2 이달의 책 관리

홈페이지의 "이달의 책 추천" 섹션을 관리합니다.

#### 관리 페이지 이동
1. 이달의 책 섹션의 **관리** 버튼을 클릭합니다.
2. 또는 `/admin/programs` 페이지로 직접 이동합니다.

> **참고**: v2.0부터 홈페이지에서 직접 책을 추가/삭제하는 기능이 제거되었습니다.
> 모든 책 관리는 **프로그램 관리 페이지**에서 수행합니다.

### 3.3 실시간 연동 확인

홈페이지에 표시되는 데이터는 다음과 연동됩니다:
- **이번주 독모**: 프로그램 관리의 주차별 도서 배정
- **이달의 독토**: 독토 페이지의 토론회 목록 (최대 4개)
- **실시간 Talk**: Talk 페이지의 게시물 (최대 3개)

## 4. 프로그램 관리

### 4.1 관리 페이지 접근

- URL: `/admin/programs`
- 또는 홈페이지 → 관리 버튼 → 프로그램 관리

### 4.2 이달의 책 관리

#### 4.2.1 책 추가
1. "책 추가" 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **도서명** (필수)
   - **저자** (필수)
   - **표지 이미지 URL**
   - **카테고리**
3. "추가" 버튼을 클릭합니다.

#### 4.2.2 책 수정
1. 책 목록에서 수정할 책의 **편집 아이콘**을 클릭합니다.
2. 정보를 수정합니다.
3. "저장" 버튼을 클릭합니다.

#### 4.2.3 책 삭제
1. 책 목록에서 삭제할 책의 **삭제 아이콘**을 클릭합니다.
2. 확인 대화상자에서 "삭제"를 클릭합니다.

### 4.3 주차별 도서 배정

독모에서 읽을 도서를 주차별로 배정합니다.

#### 배정 방법
1. **주차 선택**: 배정할 주차를 선택합니다.
2. **여명독/윤슬독 도서**: 아침/점심 독모용 도서 선택
3. **달빛독 도서**: 저녁 독모용 도서 선택
4. "저장" 버튼을 클릭합니다.

#### 연동 확인
배정된 도서는 다음 위치에 자동 반영됩니다:
- 홈페이지 > 이번주 독모 카드
- 독모 상세 페이지
- 내 서재 > 나의 독모 (신청 시)

### 4.4 독모 그룹 관리

독모 그룹(여명독, 윤슬독, 달빛독)의 시간대와 장소를 관리합니다.

#### 시간대 수정
1. 그룹 설정 탭을 클릭합니다.
2. 각 그룹의 시간대를 수정합니다:
   - **여명독**: 06:00 - 09:00
   - **윤슬독**: 12:00 - 14:00
   - **달빛독**: 17:00 - 22:00
3. 장소 정보를 수정합니다.
4. "저장" 버튼을 클릭합니다.

## 5. 독토 관리

### 5.1 관리 페이지 접근

- 독토 페이지에서 관리자로 로그인 시 관리 버튼 표시
- 또는 카드에서 직접 수정/삭제

### 5.2 독토 생성

#### 생성 방법
1. 독토 페이지 상단의 **독토 생성** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **독토 제목** (필수): 토론 주제
   - **도서명** (필수): 토론할 책
   - **리더** (필수): 진행자 이름
   - **리더 유형**: 학생 / 교수 / 작가
   - **최대 인원**: 참여 가능 인원
   - **일정**: 모임 시간 (예: "매주 화 20:00")
   - **태그**: 쉼표로 구분 (예: "에세이, 자기탐색")
   - **도서 표지 URL** (선택)
   - **배경 이미지 URL** (선택)
3. "생성" 버튼을 클릭합니다.

### 5.3 독토 수정

1. 독토 카드에 마우스를 올리면 수정 아이콘이 표시됩니다.
2. **연필 아이콘**을 클릭합니다.
3. 정보를 수정합니다.
4. "수정" 버튼을 클릭합니다.

### 5.4 독토 삭제

1. 독토 카드에서 **휴지통 아이콘**을 클릭합니다.
2. 확인 대화상자에서 "삭제"를 클릭합니다.

> **주의**: 삭제된 독토는 복구할 수 없습니다.
> 참여자가 있는 독토는 신중하게 삭제해주세요.

### 5.5 독토 데이터 연동

독토 데이터는 다음과 연동됩니다:
- 홈페이지 > 이달의 독토 (최신 4개)
- 독토 페이지 > 전체 목록
- 내 서재 > 나의 독토 (신청 시)

## 6. 공지사항 관리

### 6.1 관리 페이지 접근

- URL: `/admin/notices`

### 6.2 공지 생성

1. "공지 작성" 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **제목** (필수)
   - **유형**: 공지 / 이벤트 / 업데이트
   - **내용** (필수): 마크다운 지원
   - **고정 여부**: 상단 고정 옵션
3. "게시" 버튼을 클릭합니다.

### 6.3 공지 유형별 색상

| 유형 | 색상 | 용도 |
|------|------|------|
| **공지** | 초록색 | 일반 공지사항 |
| **이벤트** | 주황색 | 이벤트, 행사 안내 |
| **업데이트** | 파란색 | 서비스 업데이트 |

### 6.4 공지 수정/삭제

- **수정**: 공지 목록에서 편집 아이콘 클릭
- **삭제**: 공지 목록에서 삭제 아이콘 클릭 후 확인

## 7. 서평 관리

### 7.1 관리 페이지 접근

- URL: `/admin/reviews`

### 7.2 서평 목록 확인

관리자는 모든 사용자의 서평을 확인할 수 있습니다:
- 작성자 정보
- 도서 정보
- 별점 및 내용
- 작성 일시
- 좋아요/댓글 수

### 7.3 서평 삭제

부적절한 서평을 삭제할 수 있습니다:
1. 서평 목록에서 삭제할 서평을 찾습니다.
2. **삭제** 버튼을 클릭합니다.
3. 삭제 사유를 입력합니다 (선택).
4. "삭제" 버튼을 클릭합니다.

> **주의**: 서평 삭제 시 해당 사용자의 스탬프와 뱃지에 영향을 줄 수 있습니다.

## 8. 고전 100선 관리

### 8.1 관리 페이지 접근

- URL: `/admin/classics`

### 8.2 도서 추가

1. "도서 추가" 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **도서명** (필수)
   - **저자** (필수)
   - **카테고리**: 문학·예술 / 인문·사회 / 자연과학
   - **출판사**
   - **출판연도**
   - **ISBN**
   - **표지 이미지 URL**

3. "추가" 버튼을 클릭합니다.

### 8.3 도서 수정/삭제

- **수정**: 도서 목록에서 편집 아이콘 클릭
- **삭제**: 도서 목록에서 삭제 아이콘 클릭 후 확인

### 8.4 연도별 관리

고전 100선은 연도별로 관리됩니다:
1. 상단의 연도 선택기에서 관리할 연도를 선택합니다.
2. 해당 연도의 도서 목록을 관리합니다.
3. 새 연도 추가 시 "연도 추가" 버튼을 클릭합니다.

## 9. 데이터 연동 시스템

### 9.1 SharedDataContext 개요

v2.0부터 도입된 전역 상태 관리 시스템입니다:

```
SharedDataContext
├── readingGroups (독모 그룹)
├── discussions (독토 목록)
├── talkPosts (Talk 게시물)
├── joinedDoktoClubs (신청한 독토)
└── appliedDokmoSessions (신청한 독모)
```

### 9.2 데이터 흐름

#### 독모 데이터
```
프로그램 관리 (주차별 도서)
    ↓
홈페이지 (이번주 독모)
    ↓
사용자 신청
    ↓
내 서재 (나의 독모)
```

#### 독토 데이터
```
독토 관리 (생성/수정/삭제)
    ↓
홈페이지 (이달의 독토) ← 최대 4개
    ↓
독토 페이지 (전체 목록)
    ↓
사용자 신청
    ↓
내 서재 (나의 독토)
```

#### Talk 데이터
```
Talk 페이지 (게시물 작성)
    ↓
홈페이지 (실시간 Talk) ← 최대 3개
```

### 9.3 실시간 동기화

- 관리자가 데이터를 수정하면 모든 페이지에 즉시 반영됩니다.
- 사용자가 신청하면 내 서재에 즉시 반영됩니다.
- 별도의 새로고침 없이 자동 업데이트됩니다.

### 9.4 데이터 일관성 유지

관리 시 주의사항:
- 독토 삭제 시 해당 독토에 참여 중인 사용자의 내 서재에서 제거됩니다.
- 도서 정보 수정 시 관련된 모든 페이지에 반영됩니다.
- 대량 수정 시 순차적으로 진행하여 시스템 부하를 방지하세요.

## 10. 자주 묻는 질문 (FAQ)

### Q1. 관리자 권한은 어떻게 부여하나요?
시스템 관리자가 계정에 관리자 권한을 부여합니다. 권한 요청은 담당자에게 문의하세요.

### Q2. 홈페이지에서 책을 직접 추가할 수 없나요?
v2.0부터 홈페이지의 직접 추가 기능이 제거되었습니다. `/admin/programs` 페이지에서 관리하세요.

### Q3. 독토를 삭제하면 참여자는 어떻게 되나요?
독토 삭제 시 참여자의 내 서재에서 해당 독토가 자동으로 제거됩니다. 중요한 독토는 삭제 전 참여자에게 공지하세요.

### Q4. 데이터가 다른 페이지에 반영되지 않아요.
SharedDataContext는 실시간 동기화를 지원합니다. 반영되지 않는 경우:
1. 페이지를 새로고침합니다.
2. 브라우저 캐시를 삭제합니다.
3. 문제가 지속되면 기술 지원에 문의하세요.

### Q5. 공지사항을 상단에 고정하려면?
공지 작성/수정 시 "고정" 옵션을 체크하면 목록 상단에 고정됩니다.

### Q6. 서평 삭제 시 스탬프도 회수되나요?
네, 서평 삭제 시 해당 서평으로 획득한 스탬프가 차감될 수 있습니다.

### Q7. 여러 관리자가 동시에 수정해도 되나요?
가능하지만, 동일한 데이터를 동시에 수정하면 충돌이 발생할 수 있습니다. 수정 전 다른 관리자와 조율하세요.
