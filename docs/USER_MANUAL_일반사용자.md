## 1. 서비스 소개

### 1.1 오거서란?

**오거서**는 성균관대학교 학생들을 위한 독서 플랫폼입니다. 함께 책을 읽고, 토론하고, 서평을 나누며 독서 문화를 즐길 수 있습니다.

### 1.2 주요 기능

| 기능 | 설명 |
|------|------|
| **독모 (독서모임)** | 시간대별 독서모임 참여 (아침/점심/저녁) |
| **독토 (독서토론회)** | 학생/교수/작가와 함께하는 독서 토론 |
| **서평 작성** | 읽은 책에 대한 리뷰 작성 및 공유 |
| **성균 고전 100선** | 추천 도서 목록 확인 |
| **내 서재** | 개인 독서 기록, 나의 독모/독토, 서평 관리 |
| **Talk Talk** | 독서 일상 공유 SNS |

### 1.3 접속 방법

웹 브라우저에서 오거서 URL에 접속합니다. 모바일과 데스크톱 모두 지원됩니다.

### 1.4 데이터 연동

오거서는 **실시간 데이터 연동**을 지원합니다:
- 홈페이지의 독모/독토/Talk 정보가 각 페이지와 동기화됩니다.
- 독모/독토 신청 시 **내 서재**에 자동으로 반영됩니다.
- 모든 페이지에서 최신 정보를 확인할 수 있습니다.

## 2. 메인 화면 (홈)

### 2.1 화면 구성

메인 화면은 다음과 같은 섹션으로 구성됩니다:

#### 2.1.1 히어로 배너
- 오거서의 메인 이미지와 슬로건이 표시됩니다.
- "함께 읽고, 함께 나누는" 메시지를 확인할 수 있습니다.

#### 2.1.2 이달의 책 추천
- 이달에 추천하는 도서 목록이 캐러셀 형태로 표시됩니다.
- 좌우 화살표로 탐색하거나 4초마다 자동 슬라이드됩니다.
- 각 책의 **제목**, **저자**, **카테고리**를 확인할 수 있습니다.
- 책 카드를 클릭하면 상세 정보 모달이 표시됩니다.

#### 2.1.3 이번주 독모
- 현재 진행 중인 독서모임이 카드 형태로 표시됩니다.
- **여명독** (아침 6-9시) - 주황색 그라데이션, 해 아이콘
- **윤슬독** (점심 12-14시) - 하늘색 그라데이션, 석양 아이콘
- **달빛독** (저녁 17-22시) - 남색 그라데이션, 달 아이콘
- 이번 주에 읽을 책 정보가 함께 표시됩니다.
- "신청하러가기" 버튼을 클릭하면 시간 선택 모달이 열립니다.

#### 2.1.4 이달의 독토
- 이달에 진행되는 독서토론회 목록입니다 (최대 4개 표시).
- 리더 유형에 따라 카드 색상이 다릅니다:
  - **학생 주도**: 초록색 그라데이션
  - **교수 주도**: 파란색 그라데이션
  - **작가 주도**: 주황색 그라데이션
- 토론 주제, 도서명, 리더 정보를 확인할 수 있습니다.

#### 2.1.5 실시간 Talk Talk
- 다른 사용자들의 독서 일상이 피드 형태로 표시됩니다 (최대 3개).
- 좋아요, 댓글 수를 확인할 수 있습니다.
- "전체 보기"를 클릭하면 Talk 페이지로 이동합니다.

### 2.2 독모 시간 신청 방법

1. 원하는 독모 카드의 "신청하러가기" 버튼을 클릭합니다.
2. 시간 선택 모달에서 원하는 시간대를 선택합니다.
3. 날짜를 선택합니다.
4. "신청하기" 버튼을 누르면 신청이 완료됩니다.
5. 신청된 독모는 **내 서재 > 나의 독모**에서 확인할 수 있습니다.

### 2.3 독토 참여 방법

1. 원하는 독토 카드를 클릭합니다.
2. 상세 정보 모달에서 토론 내용, 과제, 일정을 확인합니다.
3. "신청하기" 버튼을 누르면 참여 신청이 완료됩니다.
4. 참여 중인 독토는 **내 서재 > 나의 독토**에서 확인할 수 있습니다.

## 3. 독서모임 (독모)

### 3.1 개요

독모는 시간대별로 진행되는 독서모임입니다. 같은 책을 읽고 함께 이야기를 나눕니다.

### 3.2 독모 종류

| 독모 | 시간대 | 특징 |
|------|--------|------|
| **여명독** | 06:00 - 09:00 | 아침 독서모임, 상쾌한 시작 |
| **윤슬독** | 12:00 - 14:00 | 점심 독서모임, 여유로운 낮 |
| **달빛독** | 17:00 - 22:00 | 저녁 독서모임, 깊은 대화 |

### 3.3 신청 방법

1. 홈 화면 또는 프로그램 페이지에서 독모를 선택합니다.
2. 원하는 시간대를 클릭합니다.
3. 가능한 날짜 중 참여할 날짜를 선택합니다.
4. "신청하기" 버튼을 누릅니다.

### 3.4 이번 주 도서

- **여명독/윤슬독**: 같은 도서를 읽습니다.
- **달빛독**: 별도의 도서가 배정됩니다.
- 이번 주 도서는 홈 화면과 독모 카드에서 확인할 수 있습니다.

### 3.5 신청 확인

- 신청한 독모는 **내 서재 > 나의 독모** 섹션에서 확인할 수 있습니다.
- 그룹명, 시간, 장소, 날짜 정보가 표시됩니다.

## 4. 독서토론회 (독토)

### 4.1 개요

독토는 다양한 리더와 함께하는 독서토론회입니다. 학생, 교수, 작가가 이끄는 토론에 참여할 수 있습니다.

### 4.2 리더 유형

| 유형 | 설명 | 카드 색상 |
|------|------|----------|
| **학생 주도** | 학생이 리더로 진행 | 초록색 |
| **교수 주도** | 교수님이 리더로 진행 | 파란색 |
| **작가 주도** | 작가님이 리더로 진행 | 주황색 |

### 4.3 독토 페이지 기능

#### 4.3.1 필터링
- **전체**: 모든 독토 표시
- **학생 주도**: 학생이 이끄는 독토만 표시
- **교수 / 작가**: 교수 또는 작가가 이끄는 독토만 표시

#### 4.3.2 카드 정보
- 토론 주제 및 도서명
- 리더 정보
- 현재 참여 인원 / 최대 인원
- 다음 모임 일정
- 평점

#### 4.3.3 북마크
- 관심 있는 독토를 북마크에 추가할 수 있습니다.
- 북마크 아이콘을 클릭하면 저장/해제됩니다.

### 4.4 참여 방법

1. 독토 페이지에서 원하는 토론회 카드를 클릭합니다.
2. 상세 정보 모달에서 다음을 확인합니다:
   - 리더 소개 및 메시지
   - 토론 주제
   - 일정
   - 사전 과제
3. "신청하기" 버튼을 클릭합니다.
4. 신청 완료 후 카드에 "신청 완료" 표시가 나타납니다.

### 4.5 신청 확인

- 참여 중인 독토는 **내 서재 > 나의 독토** 섹션에서 확인할 수 있습니다.
- 도서 표지, 토론회명, 일정 정보가 표시됩니다.

## 5. 독서리뷰 (서평)

### 5.1 개요

읽은 책에 대한 서평을 작성하고 다른 사용자들과 공유할 수 있습니다. 서평 작성 시 KDC 뱃지를 획득하고 스탬프를 모을 수 있습니다.

### 5.2 서평 페이지 구성

#### 5.2.1 다른 서평 보기
- 다른 사용자들의 서평을 피드 형태로 확인할 수 있습니다.
- 작성자, 도서 정보, 별점, 서평 내용이 표시됩니다.
- 좋아요와 댓글 기능을 사용할 수 있습니다.

#### 5.2.2 KDC 분류별 필터
- 전체 / 총류 / 철학 / 종교 / 사회과학 / 자연과학 / 기술과학 / 예술 / 언어 / 문학 / 역사

### 5.3 서평 작성 방법 (4단계)

#### Step 1: 도서 선택
1. 검색창에 도서명 또는 저자명을 입력합니다.
2. 검색 결과에서 원하는 책을 선택합니다.

#### Step 2: 뱃지 선택
1. 해당 도서의 KDC 분류(뱃지)를 선택합니다.
2. 선택한 뱃지에 따라 지식 레이더에 반영됩니다.

#### Step 3: 서평 작성
1. 별점(1-5점)을 선택합니다.
2. 서평 내용을 작성합니다.
3. 사진 첨부 (선택사항)

#### Step 4: 스탬프 획득
1. 서평 작성 완료 시 스탬프가 적립됩니다.
2. 스탬프 현황을 확인할 수 있습니다.

### 5.4 서평 혜택

- **KDC 뱃지**: 분류별 서평 작성 시 해당 뱃지 획득
- **스탬프 적립**: 서평 1편당 스탬프 1개
- **보상**:
  - 10개 스탬프: 스타벅스 기프티콘
  - 20개 스탬프: 도서 상품권

## 6. 도서추천 (성균 고전 100선)

### 6.1 개요

성균관대학교에서 선정한 고전 100선 도서 목록을 확인할 수 있습니다.

### 6.2 화면 구성

- **연도 선택**: 드롭다운으로 연도별 목록 확인
- **카테고리 필터**: 전체 / 문학·예술 / 인문·사회 / 자연과학
- **검색창**: 도서명 또는 저자명 검색
- **도서 목록**: 그리드 형태로 표시

### 6.3 도서 정보

각 도서 카드에서 확인할 수 있는 정보:
- 표지 이미지
- 도서명
- 저자
- 카테고리 태그
- 출판사
- 출판연도

### 6.4 상세 보기

도서 카드를 클릭하면 상세 정보를 확인할 수 있습니다:
- 도서 소개
- ISBN
- 관련 독토/독모 정보

## 8. 내 서재

### 8.1 개요

개인의 독서 활동을 한눈에 확인할 수 있는 페이지입니다. 뱃지, 서평, 신청한 독모/독토를 관리합니다.

### 8.2 화면 구성

#### 8.2.1 헤더
- 획득한 뱃지 수
- 작성한 서평 수

#### 8.2.2 지식 레이더
- KDC 10개 분류에 대한 독서 현황을 레이더 차트로 표시합니다.
- 가장 많이 읽은 분류가 하이라이트됩니다.
- "당신은 'OOO 문학' 도서를 선호합니다" 메시지 표시

#### 8.2.3 서평 스탬프
- 은행잎 모양의 금색 스탬프로 서평 작성 현황을 표시합니다.
- 20칸 중 작성한 서평 수만큼 스탬프가 채워집니다.
- **10개 달성**: 스타벅스 기프티콘
- **20개 달성**: 도서 상품권

#### 8.2.4 나의 독모 (NEW!)
- 신청한 독서모임 목록이 표시됩니다.
- 그룹별 아이콘과 색상:
  - **여명독**: 해 아이콘, 황색
  - **윤슬독**: 석양 아이콘, 주황색
  - **달빛독**: 달 아이콘, 남색
- 도서명, 시간, 장소, 날짜 정보 확인 가능
- 신청한 독모가 없으면 안내 메시지 표시

#### 8.2.5 나의 독토 (NEW!)
- 참여 중인 독서토론회 목록이 표시됩니다.
- 리더 유형별 뱃지 색상:
  - **학생 주도**: 초록색
  - **교수 주도**: 파란색
  - **작가 주도**: 주황색
- 도서 표지, 토론회명, 참여 인원, 일정 정보 확인 가능
- 참여 중인 독토가 없으면 안내 메시지 표시

#### 8.2.6 나의 서평
- 내가 작성한 서평 목록이 표시됩니다.
- **프로그램별 분류**:
  - **독모**: 초록색 뱃지
  - **독토**: 주황색 뱃지
  - **일반**: 기본 녹색 뱃지
- "전체 보기" 클릭 시 전체 서평 모달 표시
- 모달에서 프로그램별 필터링 가능 (칩스 UI)

#### 8.2.7 지식 뱃지 진열장
- KDC 10개 분류별 뱃지 현황을 표시합니다.
- 획득한 뱃지는 색상으로 표시, 미획득은 회색 처리
- 뱃지 클릭 시 해당 분류의 내 서평 목록 확인

#### 8.2.8 독서 인증서
- 하단의 "독서 인증서" 버튼을 클릭하면 인증서를 발급받을 수 있습니다.

### 8.3 데이터 연동

- **독모 신청** → 나의 독모에 자동 반영
- **독토 참여** → 나의 독토에 자동 반영
- **서평 작성** → 나의 서평, 스탬프, 뱃지에 자동 반영

## 9. Talk Talk

### 9.1 개요

독서 일상을 공유하는 SNS 기능입니다. 사진과 함께 독서 경험을 나눌 수 있습니다.

### 9.2 피드 구성

각 게시물에서 확인할 수 있는 정보:
- 작성자 프로필 (이름, 아바타)
- 작성 시간
- 게시물 내용
- 언급된 책 (있는 경우)
- 첨부 사진
- 좋아요, 댓글, 공유 수

### 9.3 게시물 작성

1. 우측 하단의 "+" 버튼을 클릭합니다.
2. 내용을 작성합니다.
3. 책 언급하기 (선택): "@책제목" 형식으로 입력
4. 사진 첨부 (선택): 이미지 아이콘 클릭
5. "게시" 버튼을 클릭합니다.

### 9.4 상호작용

- **좋아요**: 하트 아이콘 클릭
- **댓글**: 말풍선 아이콘 클릭 후 댓글 작성
- **공유**: 공유 아이콘 클릭
- **더보기**: 수정/삭제 (본인 게시물)

## 10. 자주 묻는 질문 (FAQ)

### Q1. 독모 신청은 어떻게 취소하나요?
현재 버전에서는 신청 취소 기능이 제공되지 않습니다. 참여가 어려운 경우 담당자에게 문의해주세요.

### Q2. 독토에 여러 개 참여할 수 있나요?
네, 여러 개의 독토에 동시에 참여할 수 있습니다. 일정이 겹치지 않도록 확인해주세요.

### Q3. 서평은 수정할 수 있나요?
작성한 서평은 수정 및 삭제가 가능합니다. 서평 상세 화면에서 더보기 메뉴를 이용하세요.

### Q4. 스탬프 보상은 어떻게 받나요?
10개 또는 20개 스탬프 달성 시 자동으로 보상이 지급됩니다. 내 서재에서 확인하세요.

### Q5. 뱃지는 어떻게 획득하나요?
해당 KDC 분류의 도서에 대한 서평을 작성하면 뱃지가 획득됩니다.

### Q6. 홈 화면과 각 페이지의 정보가 다른 것 같아요.
오거서는 실시간 데이터 연동을 지원합니다. 새로고침 후에도 다르다면 관리자에게 문의해주세요.

### Q7. 내 서재에서 신청한 독모/독토가 보이지 않아요.
신청 완료 후 내 서재 페이지를 새로고침 해보세요. 그래도 보이지 않으면 다시 신청해주세요.
